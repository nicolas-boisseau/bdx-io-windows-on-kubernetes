# Limitations

<v-clicks>

- Taille des images (10GB+)
- Compatibilité des versions Windows
- Performance de démarrage
- Certains composants COM+ complexes
- Besoin de licences Windows
- Limitations du networking
- Gestion des mises à jour Windows

</v-clicks>
